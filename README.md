In this repo I've done few Test Run on MODBUS/ESP32/TCPIP/W5500.

Since I've used Platformio all the files are under src folder(All individul proj.)
and if you are using the same and want to test any of project under src just edit the platformio.ini file and add build_src_filter=+<filename.cpp>

Just in case if you are using Arduino IDE and want to use any of the project under src folder, just copy the code of whichever file you want to run and paste it on Arduino IDE
